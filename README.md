## Projets

**The Last Strain**  
Un jeu d'action en 3D que j'ai développé lors d'un hackathon avec Godot Engine. J'ai intégré FMOD pour le son et même ajouté du support Arduino pour des contrôles physiques. C'était un vrai challenge mais super enrichissant.

**MonAmiPoto**  
Un assistant IA personnel qui fonctionne même hors ligne. L'idée c'était de créer quelque chose d'utilisable au quotidien, pas juste une démo.

**RealTime Code Converter**  
Un convertisseur de code qui utilise l'IA pour traduire du code d'un langage à un autre en temps réel. Pratique quand tu veux rapidement adapter du code.

**Logbook**  
Application de transcription audio en temps réel avec l'API Gladia. Utile pour prendre des notes ou retranscrire des réunions.

**MemeMotion**  
Un projet un peu fun : une app qui compare tes expressions faciales avec des memes en temps réel. Développé avec MediaPipe pour la détection faciale.

**BlitzCrank GPT**  
Un bot Discord pour League of Legends qui utilise GPT-4 pour donner des conseils et interagir avec les joueurs.
